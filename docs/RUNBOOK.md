## 本地 Mac 宿主机

当前 Mac 宿主机仅用于：

- 编辑代码和文档。
- 查看 git 状态和 diff。
- 维护项目文档。

当前 Mac 宿主机不用于：

- 构建项目。
- 运行服务。
- 生成 `build/` 目录。
- 执行压测。
